# Bank Tracker

This is to let me paste in a copy of my bank statement and see what's happening with my money

# Hipster YOLO as of Oct 2016

* React
* Babel
* Yarn
* Node v6.8.1

# To Run Tests

`yarn && npm test`

which is equivalent to

`yarn && ./node_modules/.bin/mocha`
